DVL

A constraint-based visualization language that compiles to SQL.

Describe a chart as marks, channels, scales, and layouts. DVL trains domains, solves the frame, and lowers the whole thing to queries that run in DuckDB — live, in the browser.

Try the editor → Read the docs

Declarative marks & channels

Bind columns to a mark's channels — x, y, fill, … — and let expressions, aggregates, and cross-mark references compose.

Scales & layouts solve the frame

Continuous and categorical scales, plus multi-output layouts (bars, stacks, facets, treemaps, force) that emit normalized positions a constraint solver places.

Runs in DuckDB, in the browser

Every spec lowers to SQL and executes on DuckDB-Wasm. No server — the editor and every documentation example run the current engine live.

const v = canvas(640, 400);
v.dot("penguins", {
  x: linear("flipper_length_mm"),
  y: linear("body_mass_g"),
  fill: ordinal("species"),
}).axis("x").axis("y").legend("fill");
