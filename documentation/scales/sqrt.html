sqrt

The sqrt scale trains the same continuous {lo, hi} domain as linear but applies a square-root transform before normalizing: t = (√value − √lo) / (√hi − √lo), mapped to [0, 1]. It is pow with exponent ½. The canonical use is a radius channel (r): binding sqrt makes a dot's area — not its radius — proportional to the value, which is how a bubble should encode magnitude.

const v = canvas(640, 460);
v.dot("penguins", {
  x: linear("bill_length_mm"),
  y: linear("bill_depth_mm"),
  r: sqrt("body_mass_g"),
  fill: ordinal("species"),
}).axis("x").axis("y");
return v;

A bubble chart: r binds sqrt, so perceived area reads proportional to mass.

Options

  • Fitted — sqrt("col") fits its domain now.
  • Deferred / shared — const S = sqrt(); S("col") shares; reuse to union domains. S(4000) is a datum in the domain.

ScaleOpts (second argument):

option type default meaning
shared boolean solver decides pin (true) or forbid (false) domain sharing
side "top"|"right"|"bottom"|"left" auto pin the axis side
domain domain treatment "nice" "data", "nice", "zero", "unbounded", or { explicit: [lo, hi] }
params Record<string, Expr> {} none — the exponent is fixed at ½ (use pow for another exponent)
axis AxisOptions | false drawn axis config, or false to hide

Ticks use nice-linear. Negative domain values produce NULL under √ and drop.

On a colour channel sqrt is a sequential ramp (default blues); its legend is a gradient bar.

Examples

Anchor the radius domain at zero so bubble areas are comparable to an absolute baseline.

const v = canvas(640, 460);
v.dot("penguins", {
  x: linear("bill_length_mm"),
  y: linear("bill_depth_mm"),
  r: sqrt("body_mass_g", { domain: "zero" }),
}).axis("x").axis("y");
return v;

Shared radius domain across two layers — reuse the object so both encode size on one scale.

const heavy = db.table("penguins").top(1, { col: "body_mass_g", desc: true }, { partition: ["species"] });
const v = canvas(640, 440);
const x = linear("bill_length_mm");
const y = linear("bill_depth_mm");
const r = sqrt("body_mass_g");
v.dot("penguins", { x, y, r, fill: "#cfd3dc" });
v.dot(heavy,      { x, y, r, fill: "#e15759" });
return v;

sqrt on a positional axis — a gentle compression of a right-skewed quantity.

const v = canvas(640, 460);
v.dot("penguins", {
  x: linear("bill_length_mm"),
  y: sqrt("body_mass_g"),
}).axis("x").axis("y");
return v;
