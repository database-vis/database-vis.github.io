quantize

The quantize scale is binned: it trains a continuous {lo, hi} domain (a min/max reduction, like linear) and then cuts that range into N equal-width bins, mapping each value to its bin's output. Continuous in, discrete out — the counterpart of D3's scaleQuantize. The bin index is floor(t · N) where t = (value − lo) / (hi − lo), and it indexes the output range (a categorical palette, or a colour ramp on a colour channel). N is the length of that range.

const v = canvas(640, 460);
v.dot("penguins", {
  x: linear("bill_length_mm"),
  y: linear("flipper_length_mm"),
  fill: quantize("body_mass_g"),   // mass bucketed into equal-width classes → discrete colours
}).axis("x").axis("y");
return v;

Mass sorted into equal-width classes, each drawn a distinct colour — a choropleth-style encoding rather than a smooth ramp.

Options

  • Fitted — quantize("col") fits its {lo, hi} domain now, then bins it.
  • Deferred — const S = quantize(); S("col") is callable per channel. Note that each quantize computes its own bins from its own data range — its domain does not union with others (union: null), so it is not the sharing device that continuous scales are.

ScaleOpts (second argument):

option type default meaning
shared boolean solver decides facet-level domain sharing hint
side "top"|"right"|"bottom"|"left" auto pin the axis side (positional use)
domain domain treatment "data" "data", "zero", "unbounded", or { explicit: [lo, hi] } — sets the {lo, hi} that is then split into bins (not niceable — bins are fixed)
params Record<string, Expr> {} carries the output range/palette (see below)
axis AxisOptions | false drawn axis config, or false to hide

Bin count = range length. The number of bins is the number of entries in the output range. On a colour channel that range is the resolved scheme (default blues, sampled); otherwise it is the default categorical palette (tableau10-style, 10 entries). An explicit range array overrides both — its length becomes the bin count. Because the output is discrete, the legend is a set of discrete blocks/swatches, not a gradient.

On a colour channel each bin maps to one palette entry — a classed (binned) colour encoding.

Examples

Explicit domain — fix the {lo, hi} that gets split, so bin boundaries are stable across renders.

const v = canvas(680, 460);
const p = v.dot("penguins", {
  x: linear("bill_length_mm"),
  y: linear("flipper_length_mm"),
  fill: quantize("body_mass_g", { domain: { explicit: [2500, 6500] } }),
  r: 5,
});
p.legend("fill", { placement: "right" });   // discrete class swatches
return v;

Zero-anchored bins.

const v = canvas(640, 460);
v.dot("penguins", {
  x: linear("bill_length_mm"),
  y: linear("flipper_length_mm"),
  fill: quantize("body_mass_g", { domain: "zero" }),
}).axis("x").axis("y");
return v;
