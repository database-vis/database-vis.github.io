Scales

Operators that train a domain from data and apply a normalization per row. Continuous (linear, log, sqrt, pow, symlog), categorical (ordinal, point), and binned (quantize, threshold) families; on a color channel any continuous scale is a sequential ramp and ordinal is a categorical palette (a scheme names the palette) — color is an output space, not a scale family. Domain sharing (object identity, facet defaults, shared: false); domain treatments (nice, zero, explicit). Reference: scales/.

// ONE scale object bound across two relations declares "the same quantity": the domains
// UNION and both layers share a frame — the species means land exactly on the cloud.
const bySpecies = db.table("penguins").groupby(["species"], { bill: avg("bill_length_mm"), mass: avg("body_mass_g") });
const v = canvas(640, 360);
const x = linear(), y = linear();
v.dot("penguins", { x: x("bill_length_mm"), y: y("body_mass_g"), fill: "#cfd3dc" });
v.dot(bySpecies, { x: x("bill"), y: y("mass"), r: 6, fill: ordinal("species") })
 .axis("x").axis("y").legend("fill");
return v;

Train and apply

A scale does two things. It trains a domain from your data — in SQL, in the database: a min/max for the continuous family, the distinct categories for the categorical family — and it applies a per-row normalization against that fit. Bind one to a channel directly (x: linear("bill_length_mm")), or create it bare and apply it to a column per binding (const x = linear(); … x("bill")), which is how one scale spans several relations.

The families

  • Continuous (train min/max): linear, log, sqrt, pow, symlog.
  • Categorical (train distinct): ordinal (category → range entry), point (category → centered position — a band with zero width).
  • Binned: quantize (N equal-width bins over the trained extent), threshold (authored breakpoints, no training).

Per-scale transforms, options, and tick behavior live in the reference: scales/.

Color is an output space

Color is not a scale family — the channel decides the output. The same scale emits a number on a positional channel and a color on a color channel (fill/stroke): any continuous scale on a color channel is a sequential ramp, and ordinal is a categorical palette. A scheme option names the palette: linear("mass", { scheme: "viridis" }).

Domain sharing

Reusing one scale object across bindings declares "the same quantity": the domains union into one, so the layers share an extent and an axis — that is what the example above does with x and y across the raw cloud and the per-species means. Facet panels share domains by default, keeping panels comparable; pass { shared: false } to give each panel its own per-panel domain.

Domain treatments

The trained extent can be adjusted before use:

  • nice — round the bounds outward to nice values;
  • zero — extend the domain to include 0;
  • explicit — author the bounds yourself instead of training them.

Treatments ride the binding options (see each scale's reference page for its domain option and defaults).
