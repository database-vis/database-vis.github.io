vector

The vector mark draws a stroked segment from an anchor (x, y) given a length (px) and an angle (radians, 0 = +x, increasing clockwise in screen space). Where arrow/link take two endpoints, a vector takes a POLAR delta — the natural encoding for a flow field, wind barbs, a gradient, or any per-point direction+magnitude. One row → one segment.

// A vector per penguin: anchored at (bill, mass), pointing at an angle derived from bill
// depth, its length scaled from flipper length.
const flow = db.derive("flow", `
  SELECT bill_length_mm AS x, body_mass_g AS y,
         (bill_depth_mm - 17) * 0.4 AS angle,
         (flipper_length_mm - 170) * 0.6 AS length
  FROM penguins`);
const v = canvas(640, 460);
v.vector(flow, {
  x: linear("x"), y: linear("y"),
  length: px("length"), angle: field("angle"),
  stroke: "#4e79a7",
}).axis("x").axis("y");
return v;

A scatter-shaped field where each point emits a short stroke whose direction and length vary with the bird's other measurements.

Options

vector draws pen.line(x, y, x + length·cos(angle), y + length·sin(angle)). The anchor is affine (bind a scale, layout, or pixel literal); length is a pixel magnitude and angle is a plain number in radians.

Channel Type Default Meaning
x positional-x — anchor x
y positional-y — anchor y
length pixel — segment length in px
angle number — direction in radians (0 = +x, clockwise)
stroke color — line colour
strokeWidth pixel 1 line thickness
strokeDasharray text — SVG dash pattern
opacity number 1 overall opacity
strokeOpacity number 1 stroke-only opacity
title text — per-vector hover tooltip

length is PIXELS, not data units — it does not pass through a positional scale, so vectors keep their size regardless of the domain. There is no head; compose with arrow (angle → endpoints) if you need one.

Examples

A uniform-direction field — a fixed-angle, fixed-length quill at every data point.

const v = canvas(640, 460);
v.vector("penguins", {
  x: linear("bill_length_mm"), y: linear("body_mass_g"),
  length: 12, angle: field("bill_depth_mm"),   // radians ≈ bill_depth (a spread of directions)
  stroke: "#59a14f", strokeOpacity: 0.7,
}).axis("x").axis("y");
return v;

Magnitude-encoded — length carries a measure while a categorical fill (stroke) reads species.

const flow = db.derive("flow", `
  SELECT bill_length_mm AS x, body_mass_g AS y, species,
         atan2(bill_depth_mm - 17, 5) AS angle,
         (body_mass_g - 3000) / 60 AS length
  FROM penguins`);
const v = canvas(640, 460);
v.vector(flow, {
  x: linear("x"), y: linear("y"),
  length: px("length"), angle: field("angle"),
  stroke: ordinal("species"), strokeWidth: 1.5,
}).axis("x").axis("y");
return v;

A polar clock face — vectors from a shared center, angle swept over a category index.

const spokes = db.derive("spokes", `
  SELECT category, row_number() OVER () * 0.9 AS angle
  FROM (SELECT DISTINCT category FROM products)`);
const v = canvas(400, 400);
v.vector(spokes, {
  x: px(200), y: px(200), length: 140,
  angle: field("angle"),
  stroke: ordinal("category"), strokeWidth: 3,
});
return v;

Gradient glyphs on a grid — anchor on a band/point grid, direction from a computed slope.

const grid = db.derive("grid", `
  SELECT x, y, atan2(y - 3, x - 3) AS angle FROM
  (SELECT g1.i AS x, g2.i AS y FROM range(0,6) g1(i), range(0,6) g2(i))`);
const v = canvas(420, 420);
const gx = point(), gy = point();   // deferred point scales for the grid axes
v.vector(grid, {
  x: gx("x"), y: gy("y"), length: 18, angle: field("angle"),
  stroke: "#4e79a7",
}).axis("x").axis("y");
return v;
