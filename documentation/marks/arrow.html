arrow

The arrow mark is a directed link: a stroked segment from (x1, y1) to (x2, y2) plus a head at the tip. headType picks the head shape — a filled triangle (default), a two-stroke barb, or none (a plain segment) — and headSize is the head length in px. Use it for flows, deltas (A → B), gradient fields, and FK direction. One row → one arrow.

// One arrow per species: its mean → the Gentoo mean. The self-join to a singleton is
// modelled inline with derive(); a deferred linear() shares one domain across x1 and x2.
const pairs = db.derive("pairs", `
  WITH means AS (
    SELECT species, avg(bill_length_mm) AS bill, avg(body_mass_g) AS mass
    FROM penguins GROUP BY species)
  SELECT a.species, a.bill AS x1, a.mass AS y1, g.bill AS x2, g.mass AS y2
  FROM means a, means g
  WHERE g.species = 'Gentoo' AND a.species <> 'Gentoo'`);
const v = canvas(640, 460);
const x = linear();
const y = linear();
v.arrow(pairs, {
  x1: x("x1"), y1: y("y1"), x2: x("x2"), y2: y("y2"),
  headType: "triangle", headSize: 9, stroke: "#e15759",
}).axis("x").axis("y");
return v;

Two red arrows pointing from the Adelie and Chinstrap means to the Gentoo mean.

Options

arrow uses drawArrow: it strokes the shaft (x1,y1)→(x2,y2), then draws the head at the tip oriented along the segment. A triangle is a filled polygon in the stroke colour; a barb is two short strokes; none skips the head. The head defaults to 8 px.

Channel Type Default Meaning
x1 positional-x — tail x
y1 positional-y — tail y
x2 positional-x — tip x (where the head sits)
y2 positional-y — tip y
headType shape "triangle" "triangle" | "barb" | "none"
headSize pixel 8 head length in px
stroke color — shaft + head colour
strokeWidth pixel 1 shaft thickness
strokeDasharray text — SVG dash pattern for the shaft
opacity number 1 overall opacity
strokeOpacity number 1 stroke-only opacity
title text — per-arrow hover tooltip

Head direction is computed from the segment (atan2(y2−y1, x2−x1)), so the head always follows the shaft — no separate angle channel.

Examples

Barb heads — a lighter, open head style; same geometry, headType: "barb".

const pairs = db.derive("pairs", `
  WITH means AS (SELECT species, avg(bill_length_mm) AS bill, avg(body_mass_g) AS mass
                 FROM penguins GROUP BY species)
  SELECT a.bill AS x1, a.mass AS y1, g.bill AS x2, g.mass AS y2
  FROM means a, means g WHERE g.species = 'Gentoo' AND a.species <> 'Gentoo'`);
const v = canvas(640, 460);
const x = linear(), y = linear();
v.arrow(pairs, {
  x1: x("x1"), y1: y("y1"), x2: x("x2"), y2: y("y2"),
  headType: "barb", headSize: 10, stroke: "#59a14f", strokeWidth: 2,
}).axis("x").axis("y");
return v;

Directed FK edges over a force layout — arrows show which member follows which. route() owns both endpoint borrows and clips each end to the node's boundary (a circle for dots), so the arrowHEAD lands exactly on the target node's edge instead of being buried under the disc. The optional { pad: 2 } stands the head off the boundary by 2 px for a clean gap.

const v = canvas(760, 420);
const nodes = v.dot("members", {
  ...force("follows").pick("x", "y"), r: 8, fill: ordinal("community"),
});
v.arrow("follows", {
  ...route({
    src: nodes.via(eq("follows.src", "members.id")),
    dst: nodes.via(eq("follows.dst", "members.id")),
  }, { pad: 2 }).pick("x1", "y1", "x2", "y2"),
  headType: "triangle", headSize: 7, stroke: "#b6bdc9",
});
return v;

A vector-field of deltas — one arrow per row from (x,y) to (x+dx, y+dy), using arithmetic channels to compute the tip.

const flow = db.derive("flow", `
  SELECT bill_length_mm AS x, body_mass_g AS y,
         (bill_depth_mm - 17) * 30 AS dx, (flipper_length_mm - 200) * 5 AS dy
  FROM penguins`);
const v = canvas(640, 460);
const x = linear(), y = linear();
v.arrow(flow, {
  x1: x("x"), y1: y("y"),
  x2: x(cexpr`x + dx`),
  y2: y(cexpr`y + dy`),
  headType: "barb", headSize: 6, stroke: "#4e79a7", strokeOpacity: 0.6,
}).axis("x").axis("y");
return v;

Headless connectors — headType: "none" makes arrow behave like link (useful when toggling direction on/off from data).

const v = canvas(400, 200);
const seg = db.derive("seg", `SELECT 40 AS x1, 100 AS y1, 360 AS x2, 100 AS y2`);
v.arrow(seg, { x1: px.x("x1"), y1: px.y("y1"), x2: px.x("x2"), y2: px.y("y2"),
               headType: "none", stroke: "#333", strokeWidth: 2 });
return v;
