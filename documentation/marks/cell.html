cell

The cell mark emits one <rect> per row: a heatmap tile that fills its grid slot (Observable Plot's cell). Position it categorically with a band scale on each axis and it takes its width and height from those bands — bandX gives x/w, bandY gives y/h — then fill paints it by a value. It is container-eligible, so a cell's box can also host a nested plot.

const counts = db.table("penguins").groupby(["species", "island"], { n: count() });
const v = canvas(560, 420);
v.cell(counts, {
  ...bandX("species").pick("x", "w"),
  ...bandY("island").pick("y", "h"),
  fill: ordinal("species"),
});
return v;

A species × island grid: each band supplies both a position and an extent, so every tile fills its slot; fill paints it.

Options

cell fills the bands it is placed in: bind bandX().pick("x","w") and bandY().pick("y","h") (or give explicit pixel w/h). Unbound w/h are 0 — a tile with no band and no size is invisible. Bare strings are literals; use field(...) or a scale to bind a column.

channel kind default meaning
x positional-x — column position (categorical band or scale)
y positional-y — row position
w pixel 0 tile width (from bandX's w, or a pixel literal)
h pixel 0 tile height (from bandY's h, or a pixel literal)
fill color black tile paint (the value encoding)
stroke color none outline paint
strokeWidth pixel — outline width
strokeDasharray text — SVG dash pattern
opacity number 1 whole-mark opacity
fillOpacity number 1 interior opacity
strokeOpacity number 1 outline opacity
anchor shape topleft box reference point (rarely set — the frame supplies the box corner)
title text — per-mark hover tooltip; also a via join carrier

w, h, and strokeWidth are pixel channels (w→x, h→y). For the interval form (x/x2, y/y2) use rect/bar.

Examples

Heatmap coloured by a continuous measure. color(...) is a 256-step continuous ramp; ask the fill channel for a legend to read the scale.

const counts = db.table("penguins").groupby(["species", "island"], { n: count() });
const v = canvas(560, 420);
v.cell(counts, {
  ...bandX("species").pick("x", "w"),
  ...bandY("island").pick("y", "h"),
  fill: color("n"),
}).axis("x").axis("y").legend("fill");
return v;

Padded tiles. bandX takes the D3/Vega-Lite geometry options; paddingInner opens a gutter between adjacent tiles.

const counts = db.table("penguins").groupby(["species", "island"], { n: count() });
const v = canvas(560, 420);
v.cell(counts, {
  ...bandX("species", { paddingInner: 0.05 }).pick("x", "w"),
  ...bandY("island", { paddingInner: 0.05 }).pick("y", "h"),
  fill: color("n"),
});
return v;

Categorical × continuous fill from the sales star. Category revenue as a coloured strip — one band axis, one measure on fill. The measure is modelled inline with a rollup (join + group-by).

const revenue = db.table("orders").rollup(
  db.table("products"), { left: "product_id", right: "id" },
  "category", { amount: sum({ sql: "qty * price" }) });
const v = canvas(560, 200);
v.cell(revenue, {
  ...bandX("category", { paddingInner: 0.05 }).pick("x", "w"),
  y: px(40), h: 120,           // a single strip: band width from x, an explicit pixel height
  fill: color("amount"),
}).axis("x").legend("fill");
return v;

Container — a box hosting a nested plot. A host needs a real box, so it carries width and height: bandX gives the x-extent (x,w) and explicit y/h give the vertical extent. rect and cell are both container-eligible; rect is the idiomatic host for a facet grid.

const species = db.table("penguins").groupby(["species"], {});
const v = canvas(760, 300);
v.rect(species, { ...bandX("species", { paddingInner: 0.05 }).pick("x", "w"), y: px(20), h: 236, fill: "#f5f6fa" })
 .dot("penguins",
   { x: linear("bill_length_mm"), y: linear("body_mass_g"), fill: "#4e79a7" },
   { on: eq("species", "species") });
return v;

Shared vs free scales in nested cells. As with any container, shared: false trains a domain per tile so each panel's range is its own.

const species = db.table("penguins").groupby(["species"], {});
const v = canvas(760, 300);
v.rect(species, { ...bandX("species", { paddingInner: 0.05 }).pick("x", "w"), y: px(20), h: 236, fill: "#f5f6fa" })
 .dot("penguins",
   { x: linear("bill_length_mm", { shared: false }),
     y: linear("body_mass_g",    { shared: false }), fill: "#59a14f" },
   { on: eq("species", "species") });
return v;
