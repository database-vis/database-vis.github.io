tick

The tick mark draws a short axis-oriented stroke of len px at (x, y). orient chooses the direction: "vertical" (the default — an x-axis tick, extending in +y) or "horizontal" (a y-axis tick, extending in −x). It is the primitive an axis is built from: an axis is literally a tick layer plus a text layer over the same tick table — same mark, no bespoke renderer. One row → one tick.

// A short vertical stroke at each point, coloured by species — a rug/tick plot.
const v = canvas(640, 460);
v.tick("penguins", {
  x: linear("bill_length_mm"), y: linear("body_mass_g"),
  len: 10, orient: "vertical",
  stroke: ordinal("species"),
}).axis("x").axis("y");
return v;

Each bird is a 10 px vertical dash placed at its (bill, mass), coloured by species.

Options

tick draws pen.line: for orient: "horizontal" it strokes (x, y) → (x − len, y); otherwise (x, y) → (x, y + len). Signs assume a bottom/left axis placement (top/right placement flips them during axis construction). len defaults to 4 px.

Channel Type Default Meaning
x positional-x — tick x (base)
y positional-y — tick y (base)
len pixel 4 stroke length in px
orient shape "vertical" "vertical" (extends +y) | "horizontal" (extends −x)
stroke color — tick colour
strokeWidth pixel 1 tick thickness
strokeDasharray text — SVG dash pattern
opacity number 1 overall opacity
strokeOpacity number 1 stroke-only opacity
title text — per-tick hover tooltip

Examples

A horizontal rug on the left edge — value on y, a fixed x base, orient: "horizontal".

const v = canvas(480, 460);
const y = linear("body_mass_g");
v.tick("penguins", {
  x: 40, y, len: 12, orient: "horizontal", stroke: "#4e79a7", strokeOpacity: 0.6,
}).axis("y");
return v;

A strip / one-dimensional distribution — ticks along a single axis, one per bird.

const v = canvas(640, 160);
v.tick("penguins", {
  x: linear("flipper_length_mm"), y: 80,
  len: 24, orient: "vertical", stroke: "#333", strokeOpacity: 0.4,
}).axis("x");
return v;

Category strips — a strip plot per species, ticks at each bill length, grouped by a point scale on y.

const v = canvas(640, 300);
v.tick("penguins", {
  x: linear("bill_length_mm"), y: point("species"),
  len: 14, orient: "vertical", stroke: ordinal("species"),
}).axis("x").axis("y");
return v;

Reconstructing an axis from primitives — the tick + text pair over a distinct-values table. This is exactly what .axis("x") lowers to.

const ticks = db.derive("ticks", `SELECT DISTINCT species FROM penguins`);
const v = canvas(520, 260);
const x = point();   // deferred, shared by the tick and text layers
v.tick(ticks, { x: x("species"), y: 220, len: 6, orient: "vertical", stroke: "#333" });
v.text(ticks, { x: x("species"), y: 236, text: field("species"), anchor: "middle", fill: "#333" });
return v;

Longer ticks as gridlines — one long horizontal tick per y gridline value, spanning the plot.

const grid = db.derive("grid", `SELECT unnest([3000, 4000, 5000, 6000]) AS mass`);
const v = canvas(640, 460);
const y = linear(undefined, { domain: { explicit: [3000, 6000] } });  // deferred, shared by gridlines + dots
v.tick(grid, { x: 640, y: y("mass"), len: 600, orient: "horizontal",
               stroke: "#eee", strokeWidth: 1 });
v.dot("penguins", { x: linear("bill_length_mm"), y: y("body_mass_g"), r: 3, fill: "#4e79a7" })
 .axis("x").axis("y");
return v;
