bar

The bar mark emits one <rect> per row — it is rect under a different name, with the identical channel set and container behaviour. Reach for bar when the intent is a column/bar chart: bandX places the bars along one axis and barY sizes each from the baseline. Like rect it is container-eligible (its box can host a nested plot) and accepts either offset (x,y,w,h) or interval (x,x2,y,y2) form.

const bySpecies = db.table("penguins").groupby(["species"], { mass: avg("body_mass_g") });
const v = canvas(640, 460);
v.bar(bySpecies, {
  ...bandX("species", { paddingInner: 0.2 }).pick("x", "w"),  // band position + width
  ...barY("mass").pick("y", "h"),                             // bar top + height to baseline
  fill: ordinal("species"),
});
return v;

A column chart of average body mass per species. bandX("species") gives {x, w}, barY("mass") gives {y, h} — four positional values from two multi-output layouts, one rect per bar.

Options

Identical to rect: position as offset (x,y,w,h) or interval (x,x2,y,y2). Bare strings are literals — use field(...) or a scale to bind a column.

channel kind default meaning
x positional-x — left edge (offset) or first x edge (interval)
y positional-y — top edge (offset) or first y edge (interval)
x2 positional-x — second x edge (interval form)
y2 positional-y — second y edge (interval form)
w pixel — width in pixels (offset form)
h pixel — height in pixels (offset form)
anchor shape topleft box reference point: topleft places (x, y) at the corner, center centers the box on (x, y)
fill color black interior paint
stroke color none outline paint
strokeWidth pixel — outline width
strokeDasharray text — SVG dash pattern
opacity number 1 whole-mark opacity
fillOpacity number 1 interior opacity
strokeOpacity number 1 outline opacity
title text — per-mark hover tooltip; also a via join carrier

w, h, and strokeWidth are pixel channels (w→x, h→y). anchor defaults to topleft so x/y are the box corner — what bandX/barY hand out.

Examples

Zero-based bars. barY("mass", { zero: true }) anchors the domain at the baseline (lo = min(v, 0), hi = max(v, 0)), so the smallest category is not a degenerate height-0 bar and the y-axis ticks are zero-based.

const bySpecies = db.table("penguins").groupby(["species"], { mass: avg("body_mass_g") });
const v = canvas(640, 460);
v.bar(bySpecies, {
  ...bandX("species", { paddingInner: 0.2 }).pick("x", "w"),
  ...barY("mass", { zero: true }).pick("y", "h"),
  fill: ordinal("species"),
}).axis("x").axis("y");
return v;

Coloured by a measure. Bind fill to a continuous colour ramp keyed on the same aggregate.

const bySpecies = db.table("penguins").groupby(["species"], { mass: avg("body_mass_g") });
const v = canvas(640, 460);
v.bar(bySpecies, {
  ...bandX("species", { paddingInner: 0.2 }).pick("x", "w"),
  ...barY("mass").pick("y", "h"),
  fill: color("mass"),
}).legend("fill");
return v;

Stacked bars. bandX places the bars; stackY partitions each bar's total by island and cumulates the segments (a windowed running sum) up the height.

const stacks = db.table("penguins").groupby(["species", "island"], { n: count() });
const v = canvas(640, 460);
v.bar(stacks, {
  ...bandX("species", { paddingInner: 0.2 }).pick("x", "w"),
  ...stackY({ weight: "n", group: "species", order: "island" }).pick("y", "h"),
  fill: ordinal("island"),
});
return v;

Band padding. bandX takes the D3/Vega-Lite geometry options — paddingInner (gap between bars, fraction of step), paddingOuter (space before first / after last), padding (both), align (0–1).

const bySpecies = db.table("penguins").groupby(["species"], { mass: avg("body_mass_g") });
const v = canvas(640, 460);
v.bar(bySpecies, {
  ...bandX("species", { paddingInner: 0.35, paddingOuter: 0.1, align: 0.5 }).pick("x", "w"),
  ...barY("mass").pick("y", "h"),
  fill: ordinal("species"),
});
return v;

Fitted labels inside bars. A container-style composition: bars from bandX × barY (with { zero: true }), then a text layer that borrows each bar's solved pixel box through a via and sizes its label to fit with fitText.

const bySpecies = db.table("penguins").groupby(["species"], { mass: avg("body_mass_g") });
const v = canvas(640, 460);
const bars = v.bar(bySpecies, {
  ...bandX("species", { paddingInner: 0.2 }).pick("x", "w"),
  ...barY("mass", { zero: true }).pick("y", "h"),
  fill: ordinal("species"),
});
v.text(bySpecies, {
  ...bars.via(eq("species", "species")).pick({ x: "x", y: "y", w: "w", h: "h" }),
  ...fitText(),
  text: field("species"), anchor: "middle", baseline: "middle", fill: "#fff",
});
return v;

Bar as a facet container. Because bar is container-eligible, its box can host a nested plot — the same mechanism as rect.

const species = db.table("penguins").groupby(["species"], {});
const v = canvas(760, 260);
v.bar(species, { ...eqX("species").pick("x", "w"), y: px(12), h: 236, fill: "#f5f6fa" })
 .dot("penguins",
   { x: linear("bill_length_mm"), y: linear("body_mass_g"), fill: "#4e79a7" },
   { on: eq("species", "species") });
return v;
