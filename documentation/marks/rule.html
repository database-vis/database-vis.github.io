rule

The rule mark draws a stroked line segment from (x, y) to (x2, y2). It is DVL's reference-line primitive: a horizontal or vertical rule (leave one endpoint's second coordinate unset — x2 defaults to x, y2 to y), a threshold marker, an error whisker, or a text underline. One row → one segment.

// A baseline: one horizontal rule across the plot at the mean body mass.
const stats = db.derive("stats", `SELECT avg(body_mass_g) AS mass FROM penguins`);
const v = canvas(640, 460);
const y = linear();  // deferred scale, shared by the dots and the rule
v.dot("penguins", { x: linear("bill_length_mm"), y: y("body_mass_g"), fill: "#bbb" }).axis("x").axis("y");
v.rule(stats, { x: 0, x2: 640, y: y("mass"), stroke: "#e15759", strokeWidth: 2 });
return v;

A grey scatter with a single red mean-mass baseline drawn across the full width.

Options

rule shares the segment draw with connect (drawSegment("x", "y", "x2", "y2")). If x2 is omitted it collapses to x (a vertical rule); if y2 is omitted it collapses to y (a horizontal rule). All four positional channels are affine — bind a scale, a layout output, or a pixel literal.

Channel Type Default Meaning
x positional-x — start x (affine: scale, layout, or pixel literal)
y positional-y — start y
x2 positional-x x end x (omit for a vertical rule)
y2 positional-y y end y (omit for a horizontal rule)
stroke color — line colour (scale binding or literal)
strokeWidth pixel 1 line thickness in px
strokeDasharray text — SVG dash pattern, e.g. "4 2"
opacity number 1 overall opacity
strokeOpacity number 1 stroke-only opacity
title text — per-segment hover tooltip (SVG <title>)

Bare strings on positional channels are PIXEL literals; wrap column values in field("col") or bind a scale.

Examples

A vertical rule per category — a lollipop stem from the baseline up to each species mean.

const means = db.table("penguins").groupby(["species"], { mass: avg("body_mass_g") });
const v = canvas(640, 460);
const x = point();   // deferred, shared by rule + dot
const y = linear(undefined, { domain: { explicit: [0, 6500] } });
v.rule(means, { x: x("species"), y: 0, y2: y("mass"), stroke: "#bbb", strokeWidth: 2 })
 .axis("y");
v.dot(means, { x: x("species"), y: y("mass"), r: 5, fill: ordinal("species") }).axis("x");
return v;

A horizontal rule per group — a dot plot with a whisker from zero to each mean.

const means = db.table("penguins").groupby(["species"], { bill: avg("bill_length_mm") });
const v = canvas(640, 300);
const x = linear(undefined, { domain: { explicit: [0, 60] } });
const y = point();
v.rule(means, { x: 0, x2: x("bill"), y: y("species"), stroke: "#4e79a7" }).axis("x");
v.dot(means, { x: x("bill"), y: y("species"), r: 4, fill: "#4e79a7" }).axis("y");
return v;

A dashed threshold line at a fixed data value, spanning the whole width in pixels.

const line = db.derive("line", `SELECT 4000 AS threshold`);
const v = canvas(640, 460);
const y = linear();  // deferred, shared by the dots and the threshold rule
v.dot("penguins", { x: linear("bill_length_mm"), y: y("body_mass_g"), r: 3, fill: "#4e79a7" })
 .axis("x").axis("y");
v.rule(line, { x: 0, x2: 640, y: y("threshold"), stroke: "#888", strokeDasharray: "5 4" });
return v;

A diagonal reference (the y = x identity) drawn as a single sloped segment — both endpoints bound to the same shared scale.

const ref = db.derive("ref", `SELECT 0 AS lo, 60 AS hi`);
const v = canvas(480, 480);
const s = linear();  // deferred scale, shared by both axes
v.dot("penguins", { x: s("bill_length_mm"), y: s("bill_depth_mm"), r: 3, fill: "#bbb" })
 .axis("x").axis("y");
v.rule(ref, { x: s("lo"), y: s("lo"), x2: s("hi"), y2: s("hi"), stroke: "#e15759" });
return v;

Text underlines: a rule per label, its extent borrowed from the label plot's solved box via a via join.

const cats = db.table("products").groupby(["category"], {});
const v = canvas(520, 260);
const labels = v.text(cats, { x: 20, y: point("category"), text: field("category"), anchor: "start" });
v.rule(cats, {
  ...labels.via(eq("category", "category")).pick({ x: "x", y: "y" }),
  x2: 200, stroke: "#333",
});
return v;
